# UzChina Connect Trademark Clearance Checklist

This is an operational checklist, not legal advice. Before company formation, fundraising, or trademark filing, ask a qualified trademark attorney or local agent to run a formal clearance search.

## Current Brand

- Word mark: `UzChina Connect`
- Logo direction: stylized `U + C + connection arrow`
- Business scope: vetted Central Asia Chinese business resource platform, resource publishing, matching, verification, and admin-reviewed contact unlock.

## Priority Nice Classes

- Class 35: business information, advertising, commercial matching, marketplace operations.
- Class 38: online communication and transmission services.
- Class 42: SaaS, web platform, software and technical services.
- Class 39: logistics/resource channels, if logistics matching becomes a formal product.
- Class 36: escrow, payments, guarantees, or financial services, only if those products are added later.

## Search Terms

Run exact and fuzzy searches:

- `UzChina Connect`
- `UzChina`
- `UZ China`
- `Uz-China`
- `UC Connect`
- `UzChina resource`
- Cyrillic/Uzbek/Russian transliterations if filing in Central Asia.

For the logo, search both the whole logo and the icon-only mark. The `UC` monogram direction is common, so check for similar navy/gold `UC` marks with arrows, route lines, rings, or bridge forms.

## Risk Notes

- `UzChina Connect` did not show an obvious exact-match conflict in initial public web searches.
- `UzChina` appears in some existing public use, including travel/logistics/consulting contexts, so it should not be treated as fully unique without formal search.
- `UC` is a common monogram. The production logo should keep the distinctive connection arrow and route/bridge structure rather than becoming a generic `UC`.
- Trademark risk depends on territory, class, similarity, actual goods/services, and prior use. A web search is not enough for filing confidence.

## Filing Recommendation

For MVP public launch, the current name and logo direction are acceptable as a working brand. Before formal launch in Uzbekistan, China, the US, EU, or investor-facing materials, run attorney-led clearance and file the word mark first. File the logo mark after final vector refinements are stable.
